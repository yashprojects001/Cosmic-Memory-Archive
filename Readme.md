# 🌌 Cosmic Memory Archive

![Cosmic Memory Banner](https://img.shields.io/badge/Project-Cosmic%20Memory%20Archive-0ff?style=for-the-badge\&logo=github\&logoColor=black)

> ✨ A **galaxy-inspired interactive webpage** where memories live as glowing stars in a cosmic archive.

---

## 🚀 Overview

**Cosmic Memory Archive** is a creative front-end project that visualizes memories as **stars scattered across a digital universe**.

Each star represents a moment, idea, or memory.
Clicking on a star reveals its hidden story.

Instead of browsing information in a boring list, users explore a **living galaxy of memories**.

---

## 🌠 Features

🪐 **Animated Galaxy Background**

* A dynamic starfield rendered using HTML Canvas.

⭐ **Interactive Stars**

* Stars glow and twinkle continuously.

🖱 **Clickable Memory Nodes**

* Clicking a star reveals its stored memory.

💫 **Twinkling Animation**

* Stars subtly change brightness to simulate cosmic energy.

📦 **Memory Panel**

* A futuristic popup panel displays the memory title and description.

🎨 **Minimal Cosmic UI**

* Neon glow effects inspired by sci-fi interfaces.

## 🎮 How It Works

1️⃣ The canvas generates **80 random stars** across the screen.
2️⃣ Each star is assigned a **memory object**.
3️⃣ Stars animate using a **twinkle algorithm**.
4️⃣ When a user clicks near a star:

* The system detects collision distance
* The memory is retrieved
* A popup displays the memory.
